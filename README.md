# HitroCut

> AI-powered editing assistant for Adobe Premiere Pro.

## 🚧 Project Status

HitroCut is an open-source project currently under active development.

Our goal is to build an AI-powered editing assistant that understands speech and edits videos the way a professional human editor would.

## Vision

Traditional silence removers only look at audio levels.

HitroCut understands context, speech, pauses, filler words, and natural pacing before making timeline edits.

## Planned Features

- Intelligent pause detection
- Filler word removal
- False start detection
- Natural pacing
- Optional audio crossfades
- Optional J-Cuts / L-Cuts
- Review before applying edits
- Multiple AI providers
- Bring Your Own API Key support

## Philosophy

AI suggests.

Editors decide.
